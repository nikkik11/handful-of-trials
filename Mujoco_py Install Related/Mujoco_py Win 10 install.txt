MuJoCo_py 1.5.0:
	MuJoCo_py 1.50.1.68 + MuJoCo version 1.50 (win 64)
	1. Install MuJoCo version 1.50:
		Download from https://www.roboti.us/
		Unzip the downloaded mjpro150 directory into C:\Users\xxx\.mujoco\mjpro150 and place your license key (the mjkey.txt file from your email) at C:\Users\xxx\.mujoco\mjkey.txt
	2. Install MuJoCo_py
		clone from githut using github desktop: https://github.com/openai/mujoco-py.git
		Enter the directory via Command Line:
			install using python -m pip install -U "mujoco-py<1.50.2,>=1.50.1"
			https://github.com/openai/mujoco-py/issues/285#issue-357009472
	3. Check MuJoCo_py version:
		iPython Notebook / CMD: import mujoco_py; print(mujoco_py.__version__)
		
		
MuJoCo_py 0.5.7:
	MuJoCo_py 0.5.7	 + MuJoCo version 1.31 (win 64)
	1. Install MuJoCo version 1.31:	
		Download from https://www.roboti.us/
		Unzip the downloaded mjpro131 directory into C:\Users\xxx\.mujoco\mjpro131 and place your license key (the mjkey.txt file from your email) at C:\Users\xxx\.mujoco\mjkey.txt
	2. Install MuJoCo_py
		Download source code from github openai/mujoco-py: https://github.com/openai/mujoco-py/releases/tag/0.5.7
		Unzip it and enter the directory via Command Line:
			install using pip install mujoco-py==0.5.7
	3. Modify library when encounter error stating [WinError 193] %1 is not a valid Win32 application:
		Edit mjlib.py, replacing "bin/mujoco131.lib" with "bin/mujoco131.dll" in the loader.
		Explicitly specify platname = "win" in platname_targdir.py
		https://stackoverflow.com/questions/38766267/python-binding-for-mujoco-physics-library-using-mujoco-py-package
		https://github.com/openai/mujoco-py/issues/83
		https://github.com/openai/mujoco-py/issues/167
		https://github.com/openai/mujoco-py/issues/223
		https://github.com/openai/mujoco-py/issues/192
	4. Modify C:\Program Files\Python35\Lib\site-packages\mujoco_py\mjviewer.py (To fix NullFunction Error when calling env.close())
		comment out the following lines in def finish(self):
			#if gl.glIsFramebuffer(self._fbo):
				#    gl.glDeleteFramebuffers(int(self._fbo))
			#if gl.glIsRenderbuffer(self._rbo):
				#    gl.glDeleteRenderbuffers(1, int(self._rbo))
